# 📚 Gambit Platform Documentation

Welcome to the Gambit Retail Media Platform - a revolutionary 360° campaign builder that transforms complex multi-engine advertising into simple, AI-driven conversations.

## 🎯 What is Gambit?

Gambit is a comprehensive retail media platform that enables brands and retailers to create, optimize, and manage sophisticated advertising campaigns across multiple engines:

- **🔍 Sponsored Products** - Search-driven product advertising
- **📱 Display Advertising** - Visual brand awareness campaigns  
- **📺 Digital In-store** - Connected retail environment advertising
- **🏪 Offline In-store** - Physical placement and promotional materials

## 🏗️ Platform Architecture

### Design System First Approach

Our architecture follows a **Design System First** approach with **Version-Controlled Applications**:

```
Storybook Library (Single Source of Truth)
    ↓ (consumes)
Gambit Application
    ├── Version Management (V1.0, V2.0, V3.0, etc.)
    └── Theme Selection (Gambit, Albert Heijn, Delhaize, etc.)
```

### Multi-Version Strategy

- **V1.0 Gambit** - Foundation version with core features
- **V2.0 Albert Heijn** - Retailer-specific implementation
- **V3.0 Gambit** - AI-enhanced universal version
- **V3.1 Gambit** - Latest refined implementation

### Multi-Theme Support

- **🌐 Gambit Theme** - Universal default theme
- **🇳🇱 Albert Heijn Theme** - Dutch supermarket branding
- **🇧🇪 Delhaize Theme** - Belgian/European retail branding  
- **🇺🇸 AD USA Theme** - American market branding
- **🏢 AB Theme** - Specialized segment branding

## 🎨 Component Library

This Storybook contains our complete design system:

### ✅ Available Components
- **Buttons** - Multiple variants and sizes
- **Cards** - Flexible container components

- **Navigation** - Side navigation, breadcrumbs
- **Data Display** - Tables, charts, analytics

### 🎯 Design Principles

1. **Accessibility First** - WCAG compliant components via Radix UI
2. **Theme Aware** - All components work across all themes
3. **Type Safe** - Full TypeScript support with proper interfaces
4. **Consistent** - Unified design language across all applications
