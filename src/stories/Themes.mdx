All components automatically inherit theme colors through CSS variables:

```typescript
    },
  }

// No theme-specific code needed in components
```

### Theme Context System

```typescript
interface ThemeConfig {
  name: string;
  displayName: string;
  colors: Record<string, string>;
  typography: {
    fontFamily: string;
    headingFont: string;
    bodyFont: string;
  };
  localization: {
    currency: string;
    dateFormat: string;
    numberFormat: string;
    language: string;
    region: string;
  };
  branding: {
    logo: string;
    favicon: string;
    companyName: string;
    tagline: string;
  };
}

// Theme context provides configuration throughout app
export function useTheme() {
  const context = useContext(ThemeContext);
  return context;
}
```

## ğŸ”§ Development Guidelines

### Creating Theme-Aware Components

#### âœ… Do - Use CSS Variables
```css
.component {
  background-color: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
  border: 1px solid hsl(var(--border));
}
```

#### âŒ Don't - Hardcode Colors
```css

  background-color: #3b82f6; /* Hard-coded blue */

  border: 1px solid #e5e7eb;
```
